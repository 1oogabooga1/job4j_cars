About the Project

A web application for publishing and managing car sale listings, built with Java, Spring MVC, Hibernate, Thymeleaf, and PostgreSQL.

Users can create advertisements, upload car photos, edit listing details, browse all available offers, and filter posts by special conditions such as recent publications or listings with photos.

The project demonstrates a classic server-side MVC architecture with controller → service → repository → database, session-based authentication, file storage for images, and relational mapping between posts, cars, brands, engines, and users.
